# 粉色机器猫固定锚点 Skill

## 目的
用于生成用户的“粉色机器猫 / 换色机器猫”商品图时，强制保持产品结构一致，避免被图像模型重画成其他机器人猫。

## 使用方式
每次生成粉色机器猫、紫色机器猫、蓝色机器猫或其他换色版本时，先引用本 skill 的锚点图，尤其是：

1. `00_success_full_body_cover_anchor.png`：成功整机锚点图，优先参考整体结构、配色、胸口、爪子轮子、包装盒。
2. `01_paw_wheel_bottom_anchor.png`：脚底小黑滚轮锚点。
3. `02_chest_port_and_front_paws_anchor.png`：胸口标志、Type-C 充电口、前爪结构锚点。
4. `03_full_structure_multi_view_anchor.png`：多角度真实结构锚点。
5. `04_packaging_box_anchor.png`：真实包装盒锚点。
6. `05_front_led_face_anchor.png`：正面猫脸、蓝色 LED 点阵圆眼锚点。
7. `06_action_pose_reference.png`：动作卡片姿势参考，只参考动作，不参考文字语言。

## 硬性锚点规则
后续只要出现“粉色机器猫”“紫色机器猫”“按固定锚点生成”“换色机器猫”，都必须遵守以下规则。

### 1. 产品身份
- 这是同一款四足机器猫，不是重新设计机器人猫。
- 任何风格变化、场景变化、颜色变化，都不能改变产品结构。
- 若用户说“紫色机器猫 / 蓝色机器猫”，理解为“只把透明粉色部件换成对应颜色”，不是重画结构。

### 2. 配色
- 标准版：白色 ABS 机身 + 亮粉色 / 透明粉色部件。
- 粉色不能偏紫，不能发红，不能每张图颜色不一致。
- 换色版只允许改透明粉色装饰件颜色，白色机身、灰色爪子、黑色面罩、蓝色 LED 点阵眼不变。

### 3. 猫脸
- 黑色光滑面罩。
- 蓝色 LED 点阵圆眼，必须是由小蓝灯点组成的圆形眼睛。
- 不要实心卡通眼睛，不要绿色眼睛，不要普通表情脸。
- 脸部外形按真实机器猫，不要改成其他机器人头型。

### 4. 头部和耳朵
- 猫耳形状按真实产品。
- 耳朵内侧有粉色边框和十字元素。
- 耳侧圆形透明粉色装饰件保留。
- 不要加鼻子、嘴巴、猫胡子等真实猫脸结构。

### 5. 胸口和充电口
- 胸口必须有真实黑色胸口图标。
- MAX SCIE 标识保留在真实位置。
- Type-C 充电口必须在胸口黑色图标正下方。
- 不能把充电口放到背部、腹部下方、侧面或其他位置。

### 6. 脖子
- 脖子是亮粉色/透明粉色圈。
- 脖子正中间有一个粉色倒梯形小件，必须保留。
- 不要改成挂件、铃铛、吊坠或其他装饰。

### 7. 四肢、爪子和轮子（最高优先级）
- 爪子前端是灰色圆润分瓣脚趾。
- 每个脚底只有一个小黑色滚轮。
- 小黑滚轮从脚底露出约一半。
- 滚轮大小约占脚底长度的一半，不能变成大轮子。
- 不能漏掉滚轮。
- 不能把爪子做成猫肉垫。
- 不能做成光秃秃的白脚。
- 不能把滚轮做成整只脚那么大。
- 前爪、后爪结构必须一致。

### 8. 身体结构
- 四足猫形结构，身体比例按真实产品。
- 肩部、髋部透明粉色壳体保留。
- 灰色关节结构保留。
- 尾巴是透明粉色上翘尾巴。
- 不要变成人形机器人，不要变成两足站立玩偶，不要变成普通机器狗。

### 9. 包装盒
- 必须使用真实包装盒参考。
- 包装盒是白色盒体，正面有机器猫图，英文：REMOTE-CONTROLLED SMART AI CAT，右下角有 3+。
- 不得私自重画成其他包装盒，不得换成通用玩具盒。
- 包装盒可以缩小作为辅助元素，但不能改变外观。

## 常用生成提示词模板

### 标准粉色版
按“粉色机器猫固定锚点”生成 3:4 商品图。产品结构必须完全按锚点图：白色 ABS 机身，统一亮粉色/透明粉色部件，黑色面罩+蓝色 LED 点阵圆眼，MAX SCIE 标识和真实黑色胸口图标，Type-C 充电口在胸口图标正下方，脖子中间粉色倒梯形小件，真实爪子灰色分瓣脚趾，小黑滚轮从脚底半露且约占脚底一半长度，真实包装盒。只能改变背景、构图、卖点排版，不得改变产品结构。

### 换色版
按“粉色机器猫固定锚点”生成换色版。注意：这是换色，不是重新设计。只把透明粉色/亮粉色装饰件改成【目标颜色】，白色机身、黑色面罩、蓝色 LED 点阵眼、胸口标志、Type-C 充电口、爪子灰色分瓣脚趾、小黑滚轮半露结构、脖子倒梯形小件、包装盒结构全部保持不变。

### 动作卡片版
按“粉色机器猫固定锚点”生成动作展示图。动作参考动作锚点图，产品结构仍按真实机器猫：蓝色 LED 点阵圆眼、真实胸口标志、Type-C 口在胸口图标正下方、灰色分瓣爪、小黑滚轮半露。动作可以变化，但不能为了动作改变四肢结构，不能做不符合机器猫结构的腿部姿势。

## 禁止项
- 禁止生成大轮子。
- 禁止漏掉脚底小黑轮。
- 禁止把爪子做成猫肉垫。
- 禁止把粉色变成紫色，除非用户明确要求换色；即便换色也不能改结构。
- 禁止把蓝色 LED 点阵眼改成卡通眼、绿色眼或实心眼。
- 禁止改变胸口标志和充电口位置。
- 禁止把脖子倒梯形小件改成吊坠。
- 禁止重画包装盒。
- 禁止把四足机器猫重画成其他机器人猫。
