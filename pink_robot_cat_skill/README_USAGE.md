# Usage

For a new image task, attach the anchor images from this repository first, then use the prompt template.

Minimum anchor set:
1. 00_success_full_body_cover_anchor.png
2. 01_paw_wheel_bottom_anchor.png
3. 02_chest_port_and_front_paws_anchor.png
4. 04_packaging_box_anchor.png

For color variants, use COLOR_CHANGE_TEMPLATE.txt.

The key rule is: change only the requested background, selling point text, scene, or transparent accent color. Do not change product structure.
