# Pink Robot Cat Anchor Skill

粉色机器猫固定锚点库。

本仓库用于保存粉色机器猫商品图生成时必须固定的结构锚点、提示词模板和参考图片。

## 核心原则

只允许按需求修改背景、场景、卖点文案，或在换色版本中只修改透明彩色件颜色；产品结构不允许重画。

## 关键锚点

- 爪子前端：灰色圆润分瓣脚趾。
- 脚底轮子：小黑色滚轮，只从脚底露出一半，不能变大、不能漏掉。
- 配色：白色 ABS 机身 + 亮粉色 / 透明粉色部件，不能偏紫。
- 胸口：MAX SCIE 标识 + 黑色胸口图标固定。
- 充电口：Type-C 口固定在胸口图标正下方。
- 猫脸：黑色面罩 + 蓝色 LED 点阵圆眼。
- 脖子：中间粉色倒梯形小件固定保留。
- 包装盒：必须按真实包装盒参考，不得重画成其他包装。
